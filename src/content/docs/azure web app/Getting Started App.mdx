## Get code from azure

<Steps>
1.  Open <a target='_blank' href="https://dev.azure.com/even-matpash/hospital-interface/_git/medic-connect">medic-code repository</a> in azure.

2. Click the clone button at the top right of the page clone the code to the local computer. <Image src={cloneProject} alt={"Reports Page"}/>

3. Click Clone in VS Code. <Image src={cloneInVsCode} alt={"Reports Page"}/>

4. Open the project in the local computer.

</Steps>

## Running the code

<Code
  title="Running the code"
  lang="ec-line"
  code={`
npm install
npm run dev
`}
/>

## Deployment

<Steps>
1. Push changes the the azure repository.

2. In the 
<a target='_blank' href="https://dev.azure.com/even-matpash/hospital-interface">azure devops</a> <Image src={pipeLineSidebar} alt={"Reports Page"}/>

3. Select the appropriate node pipeline. <Image src={pipeLineList} alt={"Reports Page"}/>

4. Click the run pipeline button. <Image src={runPipelineButton} alt={"Reports Page"}/>

5. Select the appropriate branch and click the Run button at the bottom right. <Image src={pipelineSelectBranch} alt={"Reports Page"}/>

</Steps>
