## Pages

- **Home** - displays the requests of the hospital that the user is associated to and their end date has not passed yet.
- **History** - displays the requests of the hospital that the user is associated to and their end date has already passed.
- **Reports** - displays graphs of the requests of the hospital that the user is associated to according to status.
- **Create New Request** - enables the creation of a new request for the hospital that the user is associated to.
